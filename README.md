# Employee-Management-Analytics-System
This project is a SQL-based Employee Management System designed to store and analyze employee, department, and project data. The database is normalized and supports analytical queries for business insights.

## Technologies Used
- MySQL
- MySQL Workbench
- SQL

## Database Tables
- EMPLOYEE
- DEPARTMENT
- PROJECT
- WORKS_ON

## Key Features
- Employee and department management
- Project assignment and workload tracking
- Salary analysis using aggregate functions
- Advanced SQL queries using joins, subqueries, GROUP BY, HAVING, and CASE

## Sample Queries
- Employees earning above average salary
- Highest paid employee per department
- Employees not assigned to projects
- Project-wise total working hours
